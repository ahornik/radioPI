A quoi sert cette application ?

Cette application a été développée pour piloter à distance un Raspberry Pi connecté à une chaine stéréo. Vous trouverez cette application particulièrement utile si vous souhaitez écouter une radio sans mobiliser le téléphone (la communication entre le Pi et le téléphone ne sert qu'à envoyer l'URL de la radio ; le Pi est indépendant du téléphone pour la diffusion du son sur sa sortie audio).

Comment utiliser l'application ?

  • Sélectionner une catégorie de radio
  • La liste de radios se met à jour
  • Sélectionner la radio
  • Appuyer sur le bouton "play"
  • Pour régler le volume, balayer l'écran vers la droite
  • Pour la configuration détaillée voir les explications ci-dessous

Comment configurer le Pi ?

Pour pouvoir utiliser cette application, vous devrez :
  • Connecter la sortie audio du Pi à un amplificateur (chaîne stéréo, ...)
  • Connecter votre Raspberry Pi à votre réseau local
  • Connaître l'adresse tcpip de votre raspberry Pi (en utilisant par exemple la commande "ifconfig -a" dans un terminal lancé sur le Pi ; souvent de type 192.168.1.xxx)
  • Lancer sur le Pi un serveur écoutant sur le port 10000 (un exemple de serveur est donné sur l'URL suivante : http://www.hornik.fr/RadioPi)
  • Configurer l'application RadioPi sur le téléphone avec l'adresse du Pi (balayer l'écran vers la droite jusqu'à faire apparaitre la configuration, mettre à jour l'adresse du Pi et sauver. )
  • Il est possible (mais non trivial) d'éditer la liste des radios. Pour cela il faut
    • Connecter le téléphone à iTune,
    • Cliquer sur Apps
    • Dans la deuxième partie de l'écran vous pourrez voir une table contenant les application aptes au "File Sharing" (partage de fichiers)
    • Sélectionez RadioPi vous pourrez voir un fichier appelé RadioList.xml. Ce fichier contient la description des catégories, des radios et les Urls correspondantes. si vous éditez ce fichier sur votre PC (drag&drop du fichier, puis utilisation d'un éditeur de texte basique) vous pourrez changer la configuration, la liste des radios, etc ...
    Note : les catégories ne sont pas éditables e.g. si vous introduisez une nouvelle catégorie, ou si vous modifiez l'orthographe d'une catégorie, les radios de cette catégories seront ignorées. A corriger dans une version future.

Comment fonctionne RadioPi ?

L'application dispose en mémoire d'une liste non-exhaustive d'URL permettant d'écouter des Radios, triées par catégorie, et transmet ces URLs au Raspberry Pi. L'application IOS envoie au Raspberry Pi l'URL sélectionnée en appuyant sur le bouton play. Il faut avoir installé sur le Raspberry Pi, un serveur (donné à titre d'exemple sur le site www.hornik.fr). Ce serveur (qui peut facilement être écrit en python par exemple) attendra sur le port 10000, sur le réseau local et exécutera les commandes reçues.
